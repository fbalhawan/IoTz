# IoT Device Manager

## Description

A [Nest](https://github.com/nestjs/nest) application to integrate with your in-house IoT device.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dc:up

```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
